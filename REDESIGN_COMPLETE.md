## üéì Design System Usage

To maintain consistency when adding new pages:

1. **Use Chakra Components**: Always import from '@chakra-ui/react'
2. **Follow Color Scheme**: Use `brand.*` and `accent.*` color tokens
3. **Responsive Props**: Always include base, md, lg breakpoints
4. **Animations**: Use `motion.create()` for Framer Motion components
5. **Spacing**: Use Chakra spacing scale (4, 6, 8, 12, 16, 20, 24, etc.)
6. **Typography**: Heading and Text components with proper sizes

### Example Pattern
```jsx
import { Box, Heading, Text, Button } from '@chakra-ui/react'
import { motion } from 'framer-motion'

const MotionBox = motion.create(Box)

export default function Component() {
  return (
    <MotionBox
      initial={{ opacity: 0, y: 20 }}
      whileInView={{ opacity: 1, y: 0 }}
      viewport={{ once: true }}
      transition={{ duration: 0.6 }}
    >
      <Heading size="lg" color="brand.700" mb={4}>
        Title
      </Heading>
      <Text color={useColorModeValue('gray.600', 'gray.300')}>
        Content
      </Text>
      <Button colorScheme="accent">Action</Button>
    </MotionBox>
  )
}
```
